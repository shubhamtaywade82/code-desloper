# Skill.Fish Listing: Code Deslopper

**Title:** Code Deslopper — AI Code Cleanup & Refactor

**Tagline:** Remove AI-generated code slop, fake abstractions, and duplication from Ruby/Rails, JS, TS, React, Python & Go while preserving behavior.

**Description:**
Code Deslopper is a specialized refactoring assistant designed to combat "AI slop" — the verbose, non-idiomatic, and often redundant code generated by large language models. While AI can write code quickly, it often defaults to "enterprise" patterns that add unnecessary complexity (like one-method service objects or redundant wrapper classes).

This skill provides a structured, safety-first workflow to identify these smells and safely refactor them.

**Key Features:**
- **Semantic Analysis:** Distinguishes between necessary patterns and AI-generated boilerplate.
- **Risk Scoring:** Categorizes every change from 1 (trivial) to 5 (critical).
- **Framework Support:** Deep knowledge of Ruby on Rails (concerns, callbacks, models), React (hooks, state, props), Python, and Go.
- **Two-Phase Workflow:** Phase 1 generates a detailed smell report; Phase 2 executes approved refactors.
